# Komut satırı HTTP

Tüm kullanıcı anketlerinde ve tüm curl'ün ömrü boyunca,
[HTTP](../protocols/http.md), curl'ün desteklediği en önemli ve en sık kullanılan protokol olmuştur. Bu bölüm, etkili HTTP transferleri yapmayı ve curl ile genel oynamayı açıklar.

HTTPS, kaputun altında gerçekten aynı şey olduğu için (HTTPS, ekstra bir güvenlik TLS katmanına sahip HTTP'dir), bu çoğunlukla HTTPS için de aynı şekilde çalışır. Ayrıca özel HTTPS bölümüne bakın.
